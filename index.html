    • fcaseopen

    Case-insensitive fopen for case-sensitive filesystems

    Project maintained by OneSadCookie Hosted on GitHub Pages — Theme by mattgraham

    Use it to be lazy on OS X/iOS/Linux, or to be more compatible with Windows.

    #include "fcaseopen.h"

    Replace fopen with

    extern FILE *fcaseopen(char const *path, char const *mode);

    Replace chdir with

    extern void casechdir(char const *path);
